高级电源设置中电池的通知与报警区别

发表于 2020-10-10 更新于 2021-03-12 分类于 Windows

前言

准备将笔记本放在家里边通过 ZeroTier One 远程使用，需要设置下笔记本的高级电源设置，防止停电的时候笔记本自动关机丢失数据，但是没搞懂这几个低电量提醒的区别，遂有了这篇文章。

三种提醒级别

电源高级设置中电池的可设置项如下：

整理一下，总共有三个低电量提醒级别：

  • 电池电量低
  • 保留电池电量
  • 关键电池电量

电池电量低

其中电池电量低包括：

  • 低电池电量通知（和低电量通知暂未发现不同之处，暂定为同一个，如有知道的小伙伴可以告诉我，感谢）
  • 电池电量水平低
  • 低电量操作

低电池电量通知是在电池电量达到电池电量水平低所设置的数值时执行低电量操作并且发出通知。低电量操作可以选择关机或者睡眠或者休眠（能否休眠取决于是否开启了系统休眠功能）

关键电池电量

关键电池电量包括：

  • 关键电池电量水平
  • 关键电池电量操作

在电池电量达到关键电池电量水平所设置的数值时执行关键电池电量操作。可以选择关机或者睡眠或者休眠（能否休眠取决于是否开启了系统休眠功能）

区别

经过查询和测试发现，关键电池电量是底线，也就是说电池电量水平低所设置的值不能低于这个值。如果关键电池电量低于了电池电量水平所设置的值，会有如下提示：

所以正常设置流程为：

设置电池电量水平低的值，假设设置为 25%，那么关键电池电量可以设置为 20%，当电池电量达到 25% 时，会出现如下提示并且执行电池电量低操作，如果电池电量低操作被执行了（睡眠，关机或者休眠）则电池电量不会快速往下降，而是保持操作后的状态直到充电。

当电池电量低操作设置为不执行任何操作并且电池电量继续降低，达到 20% 时，会直接执行关键电池电量操作，睡眠，关机或者休眠，不会有任何提示。
