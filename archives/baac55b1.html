curl 导入 Postman

复制好 curl 之后，可以在命令行直接执行 curl 命令查看结果，也可以将其导入 Postman，方便后续修改请求重新发起请求。

点击 import 按钮，粘贴 curl 命令，即可自动生成一个 Postman 中正常的请求，方便后续修改请求

自动生成代码

Postman 有一个根据当前的请求自动生成代码的功能，在 Post 请求界面，点击 Code 按钮，可以打开代码片段生成窗口。
