总结

这几个网站的难度不大，但是都比较繁琐，因为要使用正则的方式来各种提取 js 代码，还要使用 Python 重新实现 js 的解码函数，属实是比较难调试。不过这里可以考虑使用 AI，AI 可以将 js 代码直接翻译成 Python 代码，还是比较方便的。

或者可以直接使用 nodejs，可以更加方便的获取 js 执行的结果，不用使用正则提取数据了。
